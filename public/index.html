Arithmetic Operations

Arithmetic operations in image processing involve performing mathematical operations on the pixel values of two or more images. These operations can be used to manipulate the images to achieve desired effects or to perform calculations.

There are four basic arithmetic operations used in image processing:

Addition Filter

The addition of two images f and g of the same size results in a new image h of the same size whose pixels are to the sum of the pixels in the original images

Subtraction Filter

Image subtraction or pixel subtraction is a process whereby the digital numeric value of one pixel or whole image is subtracted from another image. Multiplication filter

Like other image arithmetic operators, multiplication comes in two main forms. The first form takes two input images and produces an output image in which the pixel values are just those of the first image, multiplied by the values of the corresponding values in the second image.

Division filter

A digital image processing technique for increasing the contrast between features in an image by dividing the pixel values in the image by the values of corresponding pixels in a second image. Image division is normally used to identify concentrations of vegetation.

Logical Operations

Logical operations in image processing involve performing Boolean operations on the pixel values of one or more images to create a new image. These operations can be used to combine or isolate specific areas of an image, or to create binary images for further processing.

There are three basic logical operations used in image processing:

And Filter

Bit-wise conjunction of input array elements.

Or

Bit-wise disjunction of input array elements.

Xor

Bit-wise exclusive-OR operation on input array elements.

Not

Inversion of input array elements.
